# Student Study Group Finder

A web-based platform designed to help students find and join study groups. This project was developed as part of the CSC1202 course.

## 🚀 Live Demo
You can view the project live here:
**[https://musiimentaabraham02-blip.github.io/student_study/](https://musiimentaabraham02-blip.github.io/student_study/)**

## ✨ Features
- **User Authentication**: Secure sign-up and login.
- **Group Management**: Create and manage study groups.
- **Search Functionality**: Find groups based on subjects or interests.
- **Session Scheduling**: Coordinate study sessions with ease.
- **Group Communication**: Integrated chat/messaging for group members.

## 📦 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/musiimentaabraham02-blip/student_study.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
